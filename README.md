Book Library App

FEATURES:
.Home page- It where when a user access the app lands on and contains functions such as search bar and links to other pages.
.About page- contains information about the library app such as the creator and explains each function and help info contacts.
.Categories component- This categorises each book genres in to one eg. Fantasy Books,Science Books,History Books and so on.
.Read List component- This page contains books that the reader has saved to read later.
.Register component- When the user wishes to create an account they access this page and register.
.Search Bar component- When the user whishes to find a certain book they use the search bar menu this enhances the UI phase of the App.
